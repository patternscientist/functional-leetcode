# 542. 01 Matrix

## Pattern

Multi-source BFS from all zero cells to fill nearest-zero distances.

## Invariant

When a cell first receives a distance, that distance is minimal because BFS expands in increasing distance order from all zero sources.

## Code

dist0 = 0 at every zero and -1 elsewhere
q0 = all zero cells
for each popped cell, assign unseen neighbors distance + 1

## Complexity

Time: O(mn). Space: O(mn).

## Pitfall(s)

Start from all zeros. Starting BFS separately from each one-cell is much slower and obscures the invariant.
